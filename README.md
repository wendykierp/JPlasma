JPlasma is a Java port of [PLASMA](https://icl.utk.edu/plasma/)

### JPlasma 1.3

[JPlasma-1.3.jar](https://repo1.maven.org/maven2/com/github/wendykierp/JPlasma/1.3/JPlasma-1.3.jar) 

[JPlasma-1.3-javadoc.jar](https://repo1.maven.org/maven2/com/github/wendykierp/JPlasma/1.3/JPlasma-1.3-javadoc.jar) 

[JPlasma-1.3-sources.jar](https://repo1.maven.org/maven2/com/github/wendykierp/JPlasma/1.3/JPlasma-1.3-sources.jar) 

JPlasma is available on maven central as

    <dependency>
        <groupId>com.github.wendykierp</groupId>
        <artifactId>JPlasma</artifactId>
        <version>1.3</version>
    </dependency>
